# Vulnerability Assessment Report for a Live Website

## Project Overview
This project presents a professional Vulnerability Assessment conducted on a live website as part of a Cyber Security Internship Task.

The objective was to identify common web vulnerabilities, classify risks, and provide remediation steps in clear business language.

## Tools Used
- Nmap
- OWASP ZAP (Passive Scan)
- Browser Developer Tools
- Canva (Report Design)

## Scope of Testing
- Open ports and services detection
- HTTP security headers analysis
- SSL/TLS configuration review
- Common web vulnerabilities (OWASP Top 10)

## Risk Classification
All identified vulnerabilities were categorized as:
-  Low
-  Medium
-  High

- Missing Security Headers
- Information Disclosure

## Recommended Fixes

- Disable unnecessary services
